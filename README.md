# 🎮 Game Reviews Website

A simple frontend website that showcases the most popular and trending games.  
Built as a learning project to practice core frontend skills — **no frameworks, just the fundamentals**.

🔗 **Live Demo:** [alyaayahia.github.io/GAME-REVIEWS](https://alyaayahia.github.io/GAME-REVIEWS/)

---

## ✨ Features

- 🕹️ Displays trending and popular games
- 📱 Clean, responsive UI that works on all screen sizes
- 🧭 Simple and intuitive navigation

---

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) | Page structure and content |
| [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) | Custom styling and layout |
| [Bootstrap 5](https://getbootstrap.com/) | Responsive grid and UI components |
| [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) | Interactivity and dynamic behavior |

---

## 📂 Project Structure

```
/GAME-REVIEWS
├── index.html     # Main page — structure and content
├── style.css      # Custom styles on top of Bootstrap
└── script.js      # JavaScript logic and interactivity
```

## 🚀 Getting Started

### Option 1 — View Live
Just visit the live demo directly in your browser:  
👉 [https://alyaayahia.github.io/GAME-REVIEWS/](https://alyaayahia.github.io/GAME-REVIEWS/)

### Option 2 — Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/alyaayahia/GAME-REVIEWS.git

# 2. Navigate into the project folder
cd GAME-REVIEWS

# 3. Open index.html in your browser
#    (double-click it, or use a live server extension in VS Code)
```

> 💡 No build tools or installations required — it's pure HTML/CSS/JS.
