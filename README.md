# 🎮 Game Reviews Website

## 📌 Overview

This project is a simple frontend website that displays the most popular games currently.

It was built as a learning project to practice core frontend development skills without using frameworks.

---

## 🚀 Features

* Display trending/popular games
* Clean and responsive UI
* Simple and easy navigation

---

## 🛠 Tech Stack

* HTML
* CSS
* Bootstrap
* JavaScript
